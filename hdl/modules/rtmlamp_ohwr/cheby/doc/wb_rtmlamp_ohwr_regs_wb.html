  • amp_en [rw]: Amplifier Enable
    write 0: disable amplifier
    write 1: enable amplifier
  • mode [rw]: Power supply operation mode
    000: Open loop (voltage) manual control via dac
    001: Open loop (voltage) test square wave
    010: Closed loop (current) manual control via pi_sp
    011: Closed loop (current) test square wave
    100: Closed loop (current) external control
    101: Open loop (voltage) arbitrary waveform
    110: Closed loop (current) arbitrary waveform
    111: Closed loop (current) arbitrary waveform + external control
  • trig_en [rw]: Triggered mode enable
    write 0: Triggered mode disabled
    write 1: Triggered mode enabled, all set-points sent when in manual
    mode (open loop and closed loop) will only become effective
    after an external trigger is received. It is ignored when in
    the test square wave modes.
  • rst_latch_sts [rw]: Reset latched status bits
    write 0: do nothing
    write 1: Reset latched status bits (autoclear)
  • wmf_rpt_mode [rw]: Repeat the waveform samples
    write 0: Execute the wveform only once
    write 1: Repeat the waveform
  • wmf_points [rw]: Size in samples of the waveform - 1
    This field specifiy the number of samples from 1 to 1024, where
    0 means 1 sample and 1023 means 1024.
  • wmf_rate_div [rw]: Divider for the waveform sample rate
    Waveform sample rate = curr_loop_sample_rate / (2 ^ wmf_rate_div)
    0: 1
    1: 2
    2: 4
    3: 8
    ...
    15: 32768
    Waveform samples will be interpoled if wmf_rate_div > 0
